Cloning Jazzee

If you're looking for the freshest Jazzee code you will have to *git* it directly from our repository. The only tricky step is making sure you get all of the submodules too.

git clone git://github.com/Jazzee/Jazzee.git jazzee
cd jazzee
git submodule update --init --recursive
